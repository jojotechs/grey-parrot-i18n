# Grey Parrot I18n

一个为开发者提供的完整多语言管理解决方案。

## 功能特点

- 🚀 轻量级 SDK 支持多平台 (Vue, React, Flutter)
- 🎯 无需预定义语义化 key，直接使用实际文案
- 📊 可视化翻译管理面板
- 🛠 便捷的命令行工具

## 项目结构

- `packages/sdk`: SDK 实现
  - `js/core`: 基于 @vue/reactivity 的核心实现
  - `js/vue`: Vue.js 集成实现
  - `js/react`: React 集成实现
  - `flutter`: Flutter SDK 实现
- `packages/dashboard`: 翻译管理面板 (Nuxt.js)
- `packages/cli`: 命令行工具
